# منصة ذكاء المنيو — Menu Intelligence Platform

منصّة لتحليل أداء المنيو في المطاعم ومساعدة قسم البحث والتطوير والإدارة على اتخاذ قرارات أفضل لتحسين الربحية وتقليل التكاليف والهدر.

## ماذا تقدّم المنصّة

- **حساب التكلفة الحقيقية** لكل منتج ومعرفة هامش الربح الفعلي لكل صنف.
- **توصيات ذكية** قابلة للتنفيذ ومرتّبة حسب الأثر المالي: إيقاف منتج في فرع معيّن، إيقافه في ساعات الركود، استبدال مورّد أوفر، تقليل الهدر، وتحسين الربحية.
- **محاكاة لاختبار القرارات قبل تطبيقها**: تغيير سعر، استبدال مورّد، تعديل تكلفة مكوّن، أو إيقاف منتج — ورؤية النتيجة المالية فوراً.
- **متابعة الهدر والمورّدين** وتحليل أعلى المكوّنات هدراً.
- **نظرة عامة تنفيذية** بمؤشرات الأداء والرسوم البيانية.
